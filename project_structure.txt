======================================
PROJECT OVERVIEW
======================================

Intern_Ai is an AI-powered interview platform with:

BACKEND FEATURES:
  - FastAPI-based REST API
  - User authentication and authorization
  - AI interview engine with multiple evaluation modes
  - Real-time speech-to-text (Whisper integration)
  - LLM-powered interview questions and analysis
  - Interruption detection and warning system
  - Performance metrics and analysis
  - Resume parsing and intelligent question generation
  - Multi-round interview support (Technical, System Design, HR)
  - Pressure mode for stress testing

FRONTEND FEATURES:
  - React-based interactive web application
  - Real-time audio recording and streaming
  - Live interruption alerts and warnings
  - Session history and detailed feedback
  - Performance visualization and analytics
  - Voice control integration
  - Multiple interviewer personas
  - Responsive design with custom styling

KEY TECHNOLOGIES:
  Backend: Python, FastAPI, SQLite, LLM APIs, OpenAI Whisper
  Frontend: React, JavaScript, Web Audio API, CSS3
